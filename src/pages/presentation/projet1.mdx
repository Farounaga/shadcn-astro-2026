# Projet 1 – Analyse intelligente de tickets

<Alert>
  <AlertTitle>Contexte</AlertTitle>
  <AlertDescription>
    Projet réalisé dans un contexte réel d’entreprise (SYADEM) pour analyser des tickets support à grande échelle.
  </AlertDescription>
</Alert>

Ce projet combine **Ruby**, **Python**, de l’**IA**, des **embeddings vectoriels**, de la **vectorisation** et du **clustering**, dans une pipeline complète de traitement des données.

## Objectifs du projet

- Centraliser l’ensemble des tickets (API ou exports NDJSON/XML)
- Nettoyer et normaliser les données textuelles, tags et dates
- Transformer le texte en vecteurs haute dimension via embeddings
- Détecter les thèmes récurrents et les clusters
- Identifier les doublons via similarité cosinus
- Générer des résumés automatiques avec un LLM
- Fournir des statistiques claires pour prioriser les actions

## Technologies utilisées

- Ruby : extraction, normalisation, similarité, clustering, reporting
- IA : Llama (résumés), mxbai (embeddings), KMeans
- Données : XML Zendesk et API Zendesk

## Valeur ajoutée pour SYADEM

- réduction du temps d’analyse
- détection rapide des problèmes récurrents
- meilleure priorisation pour les équipes
- base évolutive vers dashboard FastAPI
