# Projet 1 – Analyse intelligente de tickets

<Alert>
  <AlertTitle>Contexte</AlertTitle>
  <AlertDescription>
    Projet réalisé dans un contexte réel d’entreprise (SYADEM) pour analyser des tickets support à grande échelle.
  </AlertDescription>
</Alert>

<Card className="mt-4">
  <CardContent className="pt-6">
    Ce projet combine **Ruby**, **Python**, de l’**IA**, des **embeddings vectoriels**, de la **vectorisation** et du **clustering**, dans une pipeline complète de traitement des données.
  </CardContent>
</Card>

<Separator className="my-6" />

## Objectifs du projet

<Card className="mt-4">
  <CardHeader><CardTitle>Objectifs du projet</CardTitle></CardHeader>
  <CardContent>
    <ul className="list-disc space-y-2 pl-6">
      <li>Centraliser l’ensemble des tickets (API ou exports NDJSON/XML)</li>
      <li>Nettoyer et normaliser les données textuelles, tags et dates</li>
      <li>Transformer le texte en vecteurs haute dimension via embeddings</li>
      <li>Détecter les thèmes récurrents et les clusters</li>
      <li>Identifier les doublons via similarité cosinus</li>
      <li>Générer des résumés automatiques avec un LLM</li>
      <li>Fournir des statistiques claires pour prioriser les actions</li>
    </ul>
  </CardContent>
</Card>

<Separator className="my-6" />

## Pipeline complet

<div className="mt-4 grid gap-4 md:grid-cols-2">
  <Card>
    <CardHeader><CardTitle>1. Extraction des tickets</CardTitle></CardHeader>
    <CardContent>Import XML/exports et enrichissement API Zendesk : id, sujet, description, commentaires, tags, dates, auteurs, rôles.</CardContent>
  </Card>
  <Card>
    <CardHeader><CardTitle>2. Normalisation & nettoyage</CardTitle></CardHeader>
    <CardContent>Suppression HTML, retrait signatures/disclaimers, filtrage des réponses automatiques, correction encodages et doublons.</CardContent>
  </Card>
  <Card>
    <CardHeader><CardTitle>3. Analyse IA</CardTitle></CardHeader>
    <CardContent>Embeddings (Ollama + mxbai-embed-large), similarité cosinus, clustering KMeans, résumés automatiques par LLM.</CardContent>
  </Card>
  <Card>
    <CardHeader><CardTitle>4. Statistiques & rapports</CardTitle></CardHeader>
    <CardContent>Histogrammes temporels, pics, stats par tag, heatmaps, exports CSV/JSON.</CardContent>
  </Card>
</div>

<Separator className="my-6" />

## Technologies utilisées

<Card className="mt-4">
  <CardHeader><CardTitle>Technologies utilisées</CardTitle></CardHeader>
  <CardContent>
    <ul className="list-disc space-y-2 pl-6">
      <li>Ruby : extraction, normalisation, similarité, clustering, reporting</li>
      <li>IA : Llama (résumés), mxbai (embeddings), KMeans</li>
      <li>Données : XML Zendesk et API Zendesk</li>
    </ul>
  </CardContent>
</Card>
